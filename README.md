# Global Superstore Sales Analysis Dashboard

## Project Overview

This project analyzes sales, profit, orders, products, and shipping performance using the Global Superstore dataset.

The dashboard was built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and Interactive Visualizations.

## Problem Statement

A multinational retail company wants to analyze sales performance across regions, categories, products, and customer segments to improve decision-making and profitability.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features

### KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

### Analysis
- Sales Trend Analysis
- Sales by Region
- Sales by Category
- Sales by Sub-Category
- Profit by Shipping Mode
- Top Products Analysis

### Interactive Filters
- Region
- Segment
- Ship Mode
- Order Date

## Key Insights

- Technology category generated the highest profit.
- Standard Class shipping contributed the highest profit.
- Phones emerged as a top-performing sub-category.
- Sales showed strong growth during year-end months.
